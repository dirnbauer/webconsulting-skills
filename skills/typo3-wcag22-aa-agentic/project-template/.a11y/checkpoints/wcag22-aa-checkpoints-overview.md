# WCAG 2.2 AA checkpoint base list

WCAG 2.2 AA requires all active Level A and Level AA success criteria. This base list contains 55 active checkpoints: 31 Level A and 24 Level AA. 4.1.1 Parsing is included only as a legacy helper in the JSON/YAML data because WCAG 2.2 removed it.

## Perceivable

### 1.1 Text Alternatives

- **1.1.1 Non-text Content** — Level A; automation: partial.

### 1.2 Time-based Media

- **1.2.1 Audio-only and Video-only (Prerecorded)** — Level A; automation: manual.
- **1.2.2 Captions (Prerecorded)** — Level A; automation: manual.
- **1.2.3 Audio Description or Media Alternative (Prerecorded)** — Level A; automation: manual.
- **1.2.4 Captions (Live)** — Level AA; automation: manual.
- **1.2.5 Audio Description (Prerecorded)** — Level AA; automation: manual.
